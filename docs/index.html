What is ChartMuseum?

ChartMuseum is an open-source Helm Chart Repository written in Go (Golang), with support for cloud storage backends, including Google Cloud Storage, Amazon S3, Microsoft Azure Blob Storage, Alibaba Cloud OSS Storage and Openstack Object Storage.

ChartMuseum is a child project under the Helm umbrella, maintained primarily by Codefresh.

Installation

$curl https://raw.githubusercontent.com/helm/chartmuseum/main/scripts/get-chartmuseum | bash

with Docker(local)

$ docker run --rm -it \
  -p 8080:8080 \
  -e DEBUG=1 \
  -e STORAGE=local \
  -e STORAGE_LOCAL_ROOTDIR=/charts \
  -v $(pwd)/charts:/charts \
  ghcr.io/helm/chartmuseum:v0.13.0
